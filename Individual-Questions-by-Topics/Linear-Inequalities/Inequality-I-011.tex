%%%%%%%%%%%% Assume a<b, c>0 %%%%%%%%%%%%

\pgfmathtruncatemacro{\a}{2}
\pgfmathtruncatemacro{\b}{5}

\pgfmathtruncatemacro{\c}{3}

\pgfmathtruncatemacro{\e}{\a-\b}
\pgfmathtruncatemacro{\d}{(\a-\b)*\c}




Solve the inequality and write the solution set in interval notation.
\[ \a x\leq \b x  \d  \]


\begin{solution}
\[\begin{split}
\a x &\leq  \b x  \d\\
\e x&\leq  \d\\
x&\geq  \c
\end{split}
\]
The solution set is $[\c, \infty)$.
\end{solution}
